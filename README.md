# react-springboot

（1）训练部分：使用面向对象的方法实现两种机器学习算法BP、SVM，并将两种算法进行封装，通过接口调用两种算法，利用已知1-5月气象数据训练2个气温预测模型，并预测后续3日气温值。

（2）测试部分：利用训练好的模型，当输入最大温度，最小温度，湿度和光照时，可以得到预测得气温。

（3）数据集中输入的数据特征为最大温度，最小温度，湿度和光照，标签项为温度。

注：这里算法效果不一定准确，可以替换其他算法
